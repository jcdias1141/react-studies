# Eventos em React

Exemplos de como trabalhar com eventos em React.

## Conceitos Aprendidos

- ✅ `onClick` - Disparado ao clicar em um elemento
- ✅ `onMouseLeave` - Disparado ao sair com o mouse de um elemento
- Outros eventos: `onChange`, `onSubmit`, `onFocus`, `onBlur`, etc.

## Arquivos

- **01-click.jsx** - Exemplos básicos de onClick e onMouseLeave
